US - NY - NYPL - Ottendorfer

This is a Zotero library lookup engine for NYPL - Ottendorfer.

Unfortunately, Zotero has removed the ability to add these lookup engines to Zotero from a user interface. You can add the following snippet to the file named locate/engines.json in your Zotero data directory. To find out where your Zotero data directory is, check your Zotero preferences, Advanced tab for the data directory location. 

	{
		"name": "NYPL - Ottendorfer",
		"_urlTemplate": "http://catalog.nypl.org/search/i?SEARCH={rft:isbn},&sortdropdown=-&searchscope=58",
		"description": "Search NYPL - Ottendorfer",
		"alias": "us.ny.nypl.ottendorfer",
		"hidden": false,
                "_urlParams": [],
		"_urlNamespaces": {
			"rft": "info:ofi/fmt:kev:mtx:journal",
			"z": "http://www.zotero.org/namespaces/openSearch#",
			"": "http://a9.com/-/spec/opensearch/1.1/"
		}
	}
